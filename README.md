# 🏔️ Homestead — Randy & Jenna's Mat-Su Valley Home Hunt

A single-page home-buying resource hub for **Randy & Jenna George**, searching for a
house around **$350,000** (range **$300K–$400K**) in the **Matanuska-Susitna Valley, Alaska**.

## What's on the page
- **Best options** — four big portals (Zillow, Realtor.com, Redfin, Homes.com), each
  pre-filtered to $300K–$400K across the Mat-Su Borough. One button opens all four.
- **Search by town** — Zillow links narrowed to Wasilla, Palmer, Big Lake, Houston,
  Sutton, Willow, and Talkeetna.
- **The glossary** — every place to search by hand: national portals, local Alaska
  brokerages (Alaska Built, RE/MAX Dynamic, Properties of Alaska, and more), plus tools
  like the Mat-Su Borough parcel lookup, a mortgage calculator, and AHFC loan programs.

## Tech
Plain static site — no build step.
- `index.html` — the page
- `css/style.css` — Alaska aurora/mountain theme, mobile-first
- `js/main.js` — starfield animation, "open all" button, scroll reveals

Deployed via **GitHub Pages**.
